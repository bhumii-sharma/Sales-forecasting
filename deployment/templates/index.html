About This App

This application allows users to upload sales data for forecasting. The system predicts future sales trends based on the provided data.

How It Works

  1. Go to the "Upload Data for Prediction" page.
  2. Select your sales dataset in CSV format.
  3. The system will analyze the data and provide a forecast.
